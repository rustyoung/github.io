centos7下mysql5.7.11单机多实例配置

安装过程

注意：在执行bin/mysqld --initialize --user=mysql时会生成登录密码

shell> groupadd mysql
shell> useradd -r -g mysql -s /bin/false mysql
shell> cd /usr/local
shell> tar zxvf /path/to/mysql-VERSION-OS.tar.gz
shell> ln -s full-path-to-mysql-VERSION-OS mysql
shell> cd mysql
shell> mkdir mysql-files
shell> chmod 750 mysql-files
shell> chown -R mysql .
shell> chgrp -R mysql .
shell> bin/mysql_install_db --user=mysql    # Before MySQL 5.7.6
shell> bin/mysqld --initialize --user=mysql # MySQL 5.7.6 and up
shell> bin/mysql_ssl_rsa_setup              # MySQL 5.7.6 and up
shell> chown -R root .
shell> chown -R mysql data mysql-files
shell> bin/mysqld_safe --user=mysql &

登入mysql修改为自己需要的密码：
shell> mysql -u root -p
mysql> SET PASSWORD FOR 'root'@'localhost' = PASSWORD('newpass');

配置过程

首先执行下列步骤：

shell> su mysql
shell> cd /usr/local/mysql
shell> bin/mysqld --initialize --user=mysql --datadir=/data/mysql_3301/data
shell> bin/mysqld --initialize --user=mysql --datadir=/data/mysql_3301/data
shell> bin/mysql_ssl_rsa_setup --datadir=/data/mysql_3301/data/
shell> bin/mysql_ssl_rsa_setup --datadir=/data/mysql_3302/data/
shell> cd ~
shell> vim my.cnf

然后将以下内容输入my.cnf：

mysqld = /usr/local/mysql/bin/mysqld_safe
mysqladmin = /usr/local/mysql/bin/mysqladmin
user = root

[mysqld1]
socket = /tmp/mysql_3301.sock
port = 3301
pid-file = /data/mysql_3301/mysql_3301.pid
datadir = /data/mysql_3301/data/
user = mysql

[mysqld2]
socket = /tmp/mysql_3302.sock
port = 3302
pid-file = /data/mysql_3302/mysql_3302.pid
datadir = /data/mysql_3302/data/
user = mysql

然后启动：

shell> mysqld_multi --defaults-file=./my.cnf start 1,2
shell> mysql -u root --port 3301 -S /tmp/mysql_3301.sock -p
