zookeeper安装及配置

zookeeper安装

下载zookeeper：
curl -O http://mirror.bit.edu.cn/apache/zookeeper/zookeeper-3.4.11/zookeeper-3.4.11.tar.gz
安装过程：
tar zxvf zookeeper-3.4.11.tar.gz
mv zookeeper-3.4.11 /usr/local/zookeeper

//将zookeeper加到系统环境变量里
vim /etc/profile
export PATH="$PATH:/usr/local/zookpeer/bin"

cd /usr/local/zookeeper/bin
ln -s zkServer.sh zkServer
ln -s zkCli.sh zkCli

//创建一个配置文件
cd ../config
cp zoo_sample.cfg zoo.cfg

//启动zookeeper
zkServer start

zookeeper伪集群搭建

首先，创建3个不同的配置文件
cd /usr/local/zookeeper/conf
cp zoo.cfg zoo1.cfg
cp zoo.cfg zoo2.cfg
cp zoo.cfg zoo3.cfg
接着，分别修改每个配置文件

zoo1.cfg如下：

tickTime=2000 //Zookeeper服务器之间或客户端与服务器之间维持心跳的时间间隔
initLimit=10 //允许follower(相对于Leaderer言的“客户端”)连接并同步到Leader的初始化连接时间，以tickTime为单位。当初始化连接时间超过该值，则表示连接失败
syncLimit=5 //表示Leader与Follower之间发送消息时，请求和应答时间长度。如果follower在设置时间内不能与leader通信，那么此follower将会被丢弃
maxClientCnxns=100 //限制连接到Zookeeper的客户端数量
dataDir=/tmp/zk/data_1 //存储内存中数据库快照的位置
clientPort=2281 //监听客户端连接的端口
dataLogDir=/tmp/zk/logs_1 //事务日志目录
server.0=localhost:2287:3387 //0是服务器编号，localhost是服务器地址，2287是leader选举的端口，3387是服务器之间通信的端口
server.1=localhost:2288:3388
server.2=localhost:2289:3389

zoo2.cfg如下：

tickTime=2000
initLimit=10
syncLimit=5
dataDir=/tmp/zk/data_2
clientPort=2282
dataLogDir=/tmp/zk/logs_2
server.0=localhost:2287:3387
server.1=localhost:2288:3388
server.2=localhost:2289:3389

zoo3.cfg如下：

tickTime=2000
initLimit=10
syncLimit=5
dataDir=/tmp/zk/data_3
clientPort=2283
dataLogDir=/tmp/zk/logs_3
server.0=localhost:2287:3387
server.1=localhost:2288:3388
server.2=localhost:2289:3389
然后，分别启动

myid的数值与配置文件server后面的为基准

cd /tmp/zk/data_1
echo 0 > myid

cd /tmp/zk/data_2
echo 1 > myid

cd /tmp/zk/data_3
echo 2 > myid

cd /usr/local/zookpeer/conf
zkServer start zoo1.cfg
zkServer start zoo2.cfg
zkServer start zoo3.cfg

注意：

启动后如果发现服务并没有启动，打开zookeeper.out查找原因
查看状态
zkServer status zoo1.cfg
    ZooKeeper JMX enabled by default
    Using config: /usr/local/zookeeper/bin/../conf/zoo1.cfg
    Mode: follower

zkServer status zoo2.cfg
    ZooKeeper JMX enabled by default
    Using config: /usr/local/zookeeper/bin/../conf/zoo2.cfg
    Mode: leader

zkServer status zoo3.cfg
    ZooKeeper JMX enabled by default
    Using config: /usr/local/zookeeper/bin/../conf/zoo3.cfg
